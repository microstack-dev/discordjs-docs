# Introduction

Discord.js is a powerful Node.js module that allows you to interact with the Discord API using JavaScript/TypeScript. This documentation covers **Discord.js v14.25.1**, a stable and feature-rich version for building modern Discord bots.

## What Discord.js Is

Discord.js provides:
- Complete Discord API coverage
- Event-driven architecture
- Type-safe interactions
- Builder pattern for safe API usage
- Efficient caching and gateway management

## What Makes v14.25.1 Different

Version 14.25.1 represents a mature, stable release with:

- **Slash Commands First**: Primary focus on interaction-based commands
- **Builder Pattern**: Type-safe builders replace raw JSON payloads
- **Modern JavaScript**: Full ES2022+ support with async/await
- **Improved Performance**: Optimized gateway handling and caching
- **Enhanced Type Safety**: Comprehensive TypeScript definitions

## Why Choose Discord.js

- **Active Development**: Regular updates and bug fixes
- **Large Community**: Extensive resources and community support
- **Production Ready**: Used by thousands of bots in production
- **Full API Coverage**: Access to all Discord API features
- **Type Safety**: First-class TypeScript support

## Bot Capabilities

With Discord.js v14.25.1, you can build bots that:
- Respond to slash commands and interactions
- Send and receive messages
- Manage guilds and channels
- Handle user permissions and roles
- Process attachments and media
- Create modals and forms
- Implement complex workflows
